## 特性

**Light Memos** 是一个以 [Material You](https://m3.material.io/) 风格呈现的 Android 轻笔记软件，这些是目前已完成的功能和未来要做的事。

- [x] 纯 Compose 和 kotlin 编写，Material You 设计
- [x] 支持Markdown 编写
- [x] 支持图片
- [x] 支持TAG标签关联
- [x] 支持更换主题和颜色
- [ ] 数据本地备份还原
- [ ] Json 格式导出
- [ ] 数据云备份和还原
- [ ] 标签和笔记之间的关系修复
- [ ] ...

## 数据备份

**Light Memos** 是一个纯本地的应用，没有服务器，数据都在您的手机上，您也可以把数据备份到webdav云盘上，以后将支持的云盘服务：

- [ ] 坚果云
- [ ] 阿里云盘
- [ ] ...
